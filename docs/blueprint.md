# **App Name**: VibeCheck

## Core Features:

- Splash Screen: A splash screen will greet users when they open the app.
- Account Creation: A straightforward account creation process with username/email, and password fields.
- Password Validation: Secure password validation to confirm the new password matches the confirm password field.
- Login Screen: A simple username/password based login screen.
- Remember Me: A way to stay logged-in, so that users do not have to log in every time they use the app.

## Style Guidelines:

- Primary color: Electric purple (#BE29EC) for a bold, standout look.
- Background color: Dark, desaturated gray (#262729) to enhance the neon effect.
- Accent color: Neon green (#39FF14) to highlight interactive elements and calls to action.
- Modern, sans-serif font for headings, ensuring legibility and a futuristic vibe.
- Slightly smaller and lighter font weight for input labels and instructions to create contrast and hierarchy.
- Center-aligned layout for the login and create account forms to focus user attention.
- Subtle glow effects on input fields and buttons to enhance the neon aesthetic and indicate interactivity.
