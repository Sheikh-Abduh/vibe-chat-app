# CHAT-BASED APPLICATION

CHECK OUT SCREENSHOTS AND THEN VER-1 IN RELEASES

AFTER UNZIPPING, OPEN PROJECT IN COMMAND TERMINAL, AND THEN WRITE THE FIRST COMMAND;

"npm install"

THEN WRITE THE SECOND COMMAND AFTER THE FIRST HAS RUN;

"npm run dev"

THE APP WILL OPEN AT localhost:3000

IF NOT, THEN OPEN http://localhost:3000 IN BROWSER
